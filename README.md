# page

> \一个简单的vue分页

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
> 公共组件目录 page/src/components/public/
> 测试目录 page/src/components/myvue/
> 样式是自己随手加的可以自行修改 有什么不足请多指教
